# 凡人修仙传：打卡

---

## 缘起

话说在混沌虚空中，有一凡人名为**韩立**，资质平平，四属性伪灵根，本无缘踏入修仙大道。

然天道无常，机缘巧合之下，韩立得一神秘小绿瓶，可催熟灵药。自此，他以凡人之躯，踏上漫漫修仙路。

炼气、筑基、结丹、元婴、化神、炼虚、合体、大乘……每一境界的突破，皆需日复一日的苦修与积累。

韩立常自省："修仙者，首重心性。若无持之以恒之志，纵有逆天宝物，终难成正果。"

## 修仙之道

大道三千，殊途同归。

现代世人亦有修仙之法——**养成习惯，日积月累**。

- 每日阅读，积累灵石，可开智慧
- 每日锻炼，强健体魄，可筑根基
- 每日学习，精进技艺，可结金丹
- 每日反思，省察己身，可炼元婴

然凡人意志薄弱，三日打鱼，两日晒网，修仙之路往往半途而废。

## 传世宝物

幸有思源笔记，乃当代修士之**洞天福地**。

此插件仿照修仙体系，将习惯养成化作一场修仙之旅：

- **灵石**：每日打卡所得，记录你的修行积累
- **境界**：炼气期、筑基期、结丹期……一步步证道成仙
- **心愿单**：许下心愿，以灵石兑换，激励自己坚持修炼
- **修炼日历**：观其行，知其道，一眼看穿你的修行轨迹
- **成就徽章**：漫漫修仙路上的里程碑，记录你的每一次突破

## 开始修炼

- 在侧边栏点击「凡人修仙传」图标
- 点击灵石按钮添加今日灵石，开始你的修仙之旅
- 创建心愿单，用积累的灵石兑换奖励
- 查看修炼日历和成就徽章，见证自己的成长
